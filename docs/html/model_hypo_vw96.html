Information on Hypo-VW96
  • Defined in publication from von Wolffersdorff (1996)
  • Extension for intergranular strain from Niemunis and Herle (1997)
  • Expects the following sixteen material parameters:
    • 1: φc\varphi_c (rad),
    • 2: ν u,
    • 3: hsh_s (kPa),
    • 4: nHn_H,
    • 5: ed0e_{d0},
    • 6: ec0e_{c0},
    • 7: ei0e_{i0},
    • 8: αH\alpha_H,
    • 9: βH\beta_H
    • 10: mTm_T,
    • 11: mRm_R,
    • 12: RmaxR_\mathrm{max},
    • 13: βR\beta_R and
    • 14: χ\chi
  • Uses eleven state variables:
    • 1: void ratio
    • 2-10: intergranular strain h11h_{11}, h22h_{22}, h33h_{33}, h12h_{12}, h21h_{21}, h23h_{23}, h32h_{32}, h13h_{13} and h31h_{31}
    • 11: replacement stiffness
  • Void ratio should be initialized as state variable. If it is not provided, it can optionally be passed as sixteenth material parameter instead
