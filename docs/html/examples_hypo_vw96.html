Examples to use Hypo-VW96 from
Each example uses the material parameters from Hostun sand as published in Schranz (2018), p. 66, with additional ν\nu
  1. φc=0.576\varphi_c = 0.576
  2. ν=0.25\nu = 0.25
  3. hs=1000000.0h_s = 1000000.0
  4. nH=0.29n_H = 0.29
  5. ed0=0.63e_d0 = 0.63
  6. ec0=1.0e_c0 = 1.0
  7. ei0=1.15e_i0 = 1.15
  8. αH=0.13\alpha_H = 0.13
  9. βH=2.0\beta_H = 2.0
  10. mT=2.0m_T = 2.0
  11. mR=5.0m_R = 5.0
  12. Rmax=0.0001R_\mathrm{max} = 0.0001
  13. βR=2.0\beta_R = 2.0
  14. χ=6.0\chi = 6.0
All examples represent the same oedometric test: The initial stress is -10 kPa and is increased to -100 kPa, decreased to -50 kPa and increased again to -200 kPa. The initial void ratio is 0.7.
