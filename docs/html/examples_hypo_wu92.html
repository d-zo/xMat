Examples to use Hypo-Wu92 from
Each example uses the material parameters from densly packed Karlsuhe sand as published in Wu (1992), p. 22
  1. C1=−106.5C_1 = -106.5
  2. C2=−801.5C_2 = -801.5
  3. C3=−797.1C_3 = -797.1
  4. C4=1077.7C_4 = 1077.7
All examples represent the same oedometric test: The initial stress is -10 kPa and is increased to -100 kPa, decreased to -50 kPa and increased again to -200 kPa. The initial void ratio is 0.65.
